Stress Detection based on Sleeping Pattern (ML)

Chain-of-Thought-Based LLM for Spatial Analysis

LLM (LLaMA) Geospatial Reasoning MERN
  • Developed a local system using a chain-of-thought-based LLM (LLaMA) to solve geospatial reasoning tasks.
